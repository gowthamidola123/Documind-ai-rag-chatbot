# DocuMind AI – Intelligent Document Assistant

## 🚀 Overview
DocuMind AI is a smart document chatbot that allows users to upload PDFs and ask questions. It uses Retrieval-Augmented Generation (RAG) and also supports fallback mode without API.

## 🧠 Features
- 📄 Multi-PDF upload
- 🤖 AI-powered answers (with API)
- 📄 Smart fallback (without API)
- 📊 Admin dashboard
- 📜 Query logs

## 🛠️ Tech Stack
- Python
- Streamlit
- OpenAI API
- TF-IDF (custom vector search)

## ▶️ How to Run
pip install -r requirements.txt  
streamlit run app.py
